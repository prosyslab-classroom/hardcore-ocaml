# Hardcore OCaml
당신의 삶과 연구를 윤택하게 만들어 줄 프로그래밍 비급

## 환경설정
- [opam](https://opam.ocaml.org)이 설치되어 있는 환경을 가정한다.
- 아래 명령어로 필요한 패키지를 설치한다.
```console
$ opam install dune memtrace
```
