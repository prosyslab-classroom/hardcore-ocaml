# 모듈

## 목표
OCaml로 복잡한 프로그램을 짤 때 모듈과 펑터를 입맛에 맞게 다룰 수 있어야 한다.

## 구현
`src` 디렉토리 안에 있는 파일을 수정하여 모두 컴파일 되도록 하라.
`moduleN.ml`에 먼저 순서대로 도전하고 난 다음 `functorN.ml`에 도전하길 권한다.

## 규칙
- 수정하지 말도록 표시된 부분은 건드리지 않도록 한다.
- 제공된 타입 정의는 고치지 않는다.
